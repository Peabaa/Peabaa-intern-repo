# Top 10 VS Code Shortcuts

1. **Ctrl + Shift + F**: Global Search. Find specific text, variables, or classes across the workspace.
2. **Ctrl + K**: Generate or edit code inline.
3. **Ctrl + P**: Quick Open. Search for and open any file in the project quickly.
4. **Ctrl + /**: Toggle commenting lines of code.
5. **Ctrl + ` (backtick)**: Opens/Closes the integrated terminal.
6. **Ctrl + Shift + C**: Opens a brand new external command prompt window at your project's root folder.
7. **Alt + Left / Alt + Right**: Navigate back and forth through the open tabs in the workspace.
8. **Alt + Left Click on a file in the explorer**: Opens that file in a new split-screen editor right next to your current one.
9. **Ctrl + W**: Close current editor tab.
10. **Ctrl + K, then Ctrl + X**: Cleans up trailing invisible space at the end of the line.

# Which shortcuts do you think will be the most useful in your daily work?

I really appreciate Ctrl + Shift + F or the Global Search because it conveniently helps me find certain lines of code quickly and easily without having to sift through the files one by one to find it.

# How can keyboard shortcuts improve your productivity?

Keyboard shortcuts are able to help the user efficiently navigate, edit, or manage files and lines of code in a few presses of buttons rather than its manual usage. It helps shorten the time for a user to execute these commands all the while preventing the work flow from breaking.
